### 説明文
**手取りから適正家賃を、年収から借入可能額を、かんたん診断！**

「この家賃、払い続けられるかな…？」
「いくらの家が買えるんだろう？」

そんな住まいの不安を、このアプリが解決します。

■ こんな方におすすめ
・引っ越しを考えている1人暮らしの方
・同棲・結婚を控えたカップル
・住宅購入を検討中のご家族
・適正な家賃を知りたい方
・住宅ローンの借入可能額を知りたい方

■ 家賃診断の特徴
✓ 手取り収入を入力するだけ（万円単位で簡単）
✓ 1人暮らし・2人暮らしに対応
✓ 安全寄り・標準・上限寄りの3パターンで表示
✓ 手取りの23-32%を目安に計算（実生活を考慮）

【診断例】
手取り25万円（1人暮らし）→ 適正家賃 5.8〜8万円
手取り45万円（2人暮らし）→ 適正家賃 9.9〜13.5万円

■ 住宅ローン診断の特徴
✓ 年収・頭金を入力（万円単位）
✓ 返済期間：30年・35年・40年から選択
✓ 物件タイプ：戸建て・マンション対応
✓ 金利は2025年平均水準（0.6%）で計算
✓ カスタム金利入力も可能
✓ 固定資産税・管理費も含めた総住居費を表示

【診断例】
年収540万円、頭金100万円、30年返済
→ 借入可能額 約2,400〜2,664万円
→ 物件総額 約2,764万円

■ 診断の根拠
・家賃：手取りの22-32%（1人暮らしと2人暮らしで調整）
・住宅ローン：返済負担率25%（生活推奨）〜30-35%（審査上限）
・固定資産税：物件価格の約0.98%/年で計算
・マンション：管理費・修繕積立金（約2.5万円/月）を含む

■ こんな使い方
1. 引っ越し前に、無理なく払える家賃を確認
2. 同棲・結婚前に、2人でいくらの家賃が適正か相談
3. 住宅購入前に、借入可能額と月々の返済額を把握
4. 戸建てとマンション、どちらが良いか比較検討

■ 安心・安全
・入力したデータはサーバーに送信されません
・デバイス内でのみ計算を実行
・個人情報の収集は一切ありません

■ 無料で使える
・基本機能は完全無料
・診断回数に制限なし
・アプリ内課金なし

※ この診断結果は一般的な目安です。実際の家賃判断や借入判断は、ご自身の状況に応じて慎重にご検討ください。
